# Kiro IDE

基于 Visual Studio Code 构建的增强型 AI IDE，提供 Agentic AI 开发体验、Spec Mode 和 Agent Hooks 等独特功能。

## 🖥️ **核心特性**

- **Agentic AI 开发** - 与 AI 协作进行复杂的开发任务
- **Spec Mode** - 结构化的规格驱动开发方法
- **Agent Hooks** - 自动化工作流程触发器
- **丰富的插件生态** - 支持各种开发语言和工具
- **安全控制** - Autopilot 和 Supervised 模式灵活切换
