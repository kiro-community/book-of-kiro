# Kiro Autonomous Agent

前沿的自主智能体，能够独立处理开发任务，保持上下文并跨仓库协作。目前正在向 Kiro Pro、Pro+ 和 Power 用户推出预览版。

## 🤖 **核心特性**

- **自主工作能力** - 独立处理复杂开发任务
- **上下文保持** - 跨会话维护项目理解
- **跨仓库协作** - 在多个代码库间协调工作
- **沙盒执行** - 安全的隔离执行环境
- **团队协作** - 支持企业级团队工作流
