# Kiro Web

基于浏览器的 AI 开发环境，提供 Vibe 模式和 Autonomous 模式两种工作方式。在安全的沙盒环境中运行，支持 GitHub/GitLab 仓库连接，自动创建 Pull Request 供审查。目前正在向 Kiro Pro、Pro+ 和 Power 用户推出预览版。

## 🌐 **核心特性**

- **双模式开发** - Vibe 模式（交互式协作开发）和 Autonomous 模式（自主任务处理）
- **浏览器端开发** - 无需本地环境，在浏览器中完成开发工作
- **沙盒执行** - 安全隔离的执行环境，支持可配置的网络访问
- **仓库集成** - 连接 GitHub/GitLab 仓库，支持跨仓库协作
- **PR 工作流** - 自动创建 Pull Request，保持代码审查流程
