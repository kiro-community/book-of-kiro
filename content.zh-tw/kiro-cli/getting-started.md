# **Kiro CLI 安裝與介紹**

Kiro CLI 是 Kiro 平台的命令列元件，提供高度互動式的終端機程式設計體驗。它支援自訂 Agent、複雜程式碼庫功能建構、工作流程自動化和精確的錯誤分析。

## **安裝 Kiro CLI**

在 macOS 和 Linux 系統上安裝：

```bash
curl -fsSL https://cli.kiro.dev/install | bash
```

## **Kiro CLI 的功能介紹**

### **💬 開始聊天**

登入後，直接使用 `kiro-cli` 指令即可開始聊天（`kiro-cli` 是 `kiro-cli chat` 指令的縮寫）。

您可以使用 `kiro-cli chat --help` 來查看額外的參數。

進入聊天後，您可以使用 `/help` 查看工作階段中的指令。下文中所有以 `/` 開頭的指令都是在工作階段中執行的。

### **📁 引用檔案或資料夾**

在聊天視窗中，檔案名稱會有自動補全的效果，您可以按下 Tab 來進行路徑自動補全。

只要告訴 Agent 需要讀取的檔案路徑，Agent 就會自動讀取檔案或資料夾中的資訊。

除了手動引用，您還可以使用自訂智慧代理來設定 Agent 預設讀取的檔案。

### **⏹️ 打斷 Agent**

在 Agent 工作的過程中可以隨時使用 Ctrl+C 或者 Ctrl+D 來打斷 Agent 的工作。

### **💾 儲存工作階段和載入工作階段**

使用 `/save xxx` 來儲存工作階段，使用 `/load xxx.json` 來載入工作階段。

### **🔄 恢復工作階段**

如果您忘記儲存工作階段，希望能夠恢復工作階段，可以使用 `kiro-cli chat --resume` 來恢復在**目前資料夾**的上一次工作階段。

### **📝 上下文管理**

您可以使用 `/usage` 查看目前上下文使用情況，使用 `/compact` 來壓縮上下文（如果工作階段超出上下文，Kiro CLI 也會自動壓縮），使用 `/clear` 清空上下文。

### **🔧 MCP 與工具**

可以使用 `/mcp` 查看目前啟用的 MCP 伺服器，使用 `/tools` 查看目前載入了哪些工具，使用 `/prompts` 查看載入了哪些提示詞。

### **🚪 退出聊天**

以下任意方式都可以退出聊天：

- 連續按下兩次 Ctrl+C
- 連續按下兩次 Ctrl+D
- 輸入 `/quit`
- 輸入 `/exit`

### **複製圖片**
Kiro CLI 支援透過 /paste 指令貼上剪貼簿中的圖片到對話中。這是 CLI 特有功能，IDE 中可以直接拖曳圖片到聊天視窗。
使用方式：
```
1. 先複製圖片到剪貼簿
2. 在 CLI 聊天中輸入 /paste
3. 圖片會被作為多模態輸入發送給模型
```
