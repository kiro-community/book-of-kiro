# **Kiro IDE 快速鍵完全掌握指南**

## 🎯 **必記！超實用的 Kiro 快速鍵 TOP10**

只要先記住這幾個，基本上你就能用好 Kiro 的 80% 功能了：

| 想做的事                            | Mac 快速鍵           | Windows / Linux 快速鍵 | 會發生什麼？                           |
| ----------------------------------- | -------------------- | ---------------------- | -------------------------------------- |
| 把選中的程式碼新增到新的對話中 | Cmd + L              | Ctrl + L               | 將選中的程式碼直接新增到新的對話中，開始和 AI 對話           |
| 把選中的程式碼新增到目前對話          | Cmd + Shift + L      | Ctrl + Shift + L       | 將選中的程式碼直接新增到目前開啟的對話中 |
| 選中程式碼精準編輯                    | Cmd + I              | Ctrl + I               | 在游標處啟動 AI，可以邊寫程式碼邊提問    |
| 命令面板                            | Cmd + Shift + P      | Ctrl + Shift + P       | 開啟命令面板，存取 Kiro 所有功能       |
| 開啟檔案                            | Cmd + P              | Ctrl + P               | 輸入檔案名稱的一部分就能快速開啟         |
| 儲存檔案                            | Cmd + S              | Ctrl + S               | 老朋友了，必須記                       |
| 專案內搜尋                          | Cmd + Shift + F      | Ctrl + Shift + F       | 在整個專案裡搜尋文字內容               |
| 跳轉到定義                          | F12                  | F12                    | 一鍵跳轉到函式或類別的定義處             |
| 修復錯誤                            | 懸停錯誤 → Quick fix | 相同                   | AI 會自動給出修復建議                  |
| 開啟終端機                            | Ctrl + （反引號 \`） | Ctrl + （反引號 \`）   | 顯示或隱藏整合終端機                     |
| 顯示/隱藏側邊欄                     | Cmd + B              | Ctrl + B               | 顯示或隱藏側邊欄，適合想全螢幕操作時使用 |
## **💡 實用又方便的小技巧快速鍵**

```bash
# 多游標編輯（學會它效率翻 3 倍）
Option + 點擊（Mac）/ Alt + 點擊（Windows）
→ 可以在多個位置同時出現游標，同時編輯多個地方

# 移動整行程式碼（超級實用的小技巧）
Option + ↑/↓（Mac）/ Alt + ↑/↓（Windows）
→ 整行程式碼上下移動，排版修改超方便

# 複製整行（寫測試資料時超好用）
Option + Shift + ↑/↓（Mac）/ Alt + Shift + ↑/↓（Windows）
→ 直接複製目前行並插入上下位置

# 快速註解（最常用，沒有之一）
Cmd + /（Mac）/ Ctrl + /（Windows）
→ 選中程式碼一鍵註解/取消註解，除錯神技！
```
