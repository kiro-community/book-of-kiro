# Kiro IDE

基於 Visual Studio Code 建構的增強型 AI IDE，提供 Agentic AI 開發體驗、Spec Mode 和 Agent Hooks 等獨特功能。

## 🖥️ **核心特性**

- **Agentic AI 開發** - 與 AI 協作進行複雜的開發任務
- **Spec Mode** - 結構化的規格驅動開發方法
- **Agent Hooks** - 自動化工作流程觸發器
- **豐富的外掛生態** - 支援各種開發語言和工具
- **安全控制** - Autopilot 和 Supervised 模式靈活切換
