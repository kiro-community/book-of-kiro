# Kiro Autonomous Agent

前沿的自主智慧代理，能夠獨立處理開發任務，保持上下文並跨儲存庫協作。目前正在向 Kiro Pro、Pro+ 和 Power 使用者推出預覽版。

## 🤖 **核心特性**

- **自主工作能力** - 獨立處理複雜開發任務
- **上下文保持** - 跨工作階段維護專案理解
- **跨儲存庫協作** - 在多個程式碼庫間協調工作
- **沙箱執行** - 安全的隔離執行環境
- **團隊協作** - 支援企業級團隊工作流程
