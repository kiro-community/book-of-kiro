# Kiro 簡介

## 📌 **Kiro 是什麼？**

Kiro 是 AWS 推出的劃時代 AI 開發平臺，包含 IDE、CLI 和 Autonomous Agent 三大元件。<br>
"哈？又一個編輯器？VSCode、Cursor 不就夠用了？"——如果你也這麼想，那請先別急著關掉頁面。因為**Kiro 真的不是普通的編輯器，而是一個完整的 Agentic AI 開發平臺。**

## 🤔 **和傳統開發工具的決定性差別**

傳統開發工具寫程式碼這件事，100% 都是你一個人在扛。<br>
但 Kiro 不一樣，它是你的 **AI 開發搭檔**，能和你一起思考、提出建議、甚至動手寫程式碼。<br>
就像有個經驗超強的前輩工程師，隨時在旁邊和你一起 pair programming 的感覺！<br>

## ✨ **Kiro 平臺能做什麼**

🤖 **和 AI 對話就能寫程式碼**：只要說一句"幫我做個登入功能"，它真的就能搞出來<br>
📋 **自動生成規格說明書**：從模糊的需求中，變出一份清清楚楚的 specs<br>
🔧 **Agent Hooks 自動化操作**：比如你一儲存檔案，它就能自動跑測試、格式化程式碼<br>
🎯 **Steering 記住專案知識點**：像"我們專案用 TypeScript 啦"這種背景，它記一次就好<br>
🔌 **MCP 支援外部工具聯動**：可以和 GitHub、AWS Docs、甚至 Web 搜尋整合使用<br>
🛡️ **安全的執行環境**：防止你一不小心"啊啊啊我把東西全刪了"的悲劇發生<br>

## 🏗️ **三大核心元件**

### 🖥️ **Kiro IDE**
基於 Visual Studio Code 構建的增強型 AI IDE，提供完整的圖形化開發體驗。

### ⚡ **Kiro CLI** 
強大的命令列工具，在終端中提供高度互動式的程式設計體驗。

### 🤖 **Kiro Autonomous Agent**
前沿的自主智慧體，能夠獨立處理開發任務並跨儲存庫協作。
